Classpublic class UnitTester
InheritanceUnitTester flash.events.EventDispatcher
Subclasses CBTester, DragAndDropScript

The test engine for unit testing Flex framework components. A UnitTester gets linked in as a mixin and when initialized finds a set of tests to run and runs them. Test scripts are written as MXML components derived from this class, and contain a bunch of TestCases with TestStep- derived child tags. They must also be [Mixin] and call setScript.
